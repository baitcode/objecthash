# ObjectHash

A content hash algorithm which works across multiple encodings (JSON, Protobufs, etc)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'objecthash'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install objecthash
